                                  Spy-Backdoor  на основе легитимного софта
                    
                                                      Представляю вашему вниманию бэкдор на основе легитимного софта -
                                                      Софт на основе Remote Utilities (RUT) Бэкдор устанавливается скрытно, закрепляясь в системе как легитимное ПО

                                                                                                                     Возможности:
                                                                                           НЕТ зависимости платформы .NET Framework !

* Работа в обход NAT
* Скрытая установка (все окна легитимной версии полностью вырезаны)
* Полная поддержка Windows XP\Vista\7\8\8.1\10 x86-64 а так же серверных систем
* Стандартный MSI не используется, ставится на прямую, тем самым не оставляя лишних следов
* UninstallerID для коннекта шлёт на email (Имя компьютера, имя пользователя, ID)
* Ветка реестра используется не стандартная. Маскируется под другую легитимную софтину, не вызывающую подозрений у юзера или системного администратора.
* Не вызывает User Account Control (UAC)
* Управление осуществляется в клиенте
* Не вызывает брандмауэр
* Portable версия клиента
* Коннект через выделенные сервера (ID отображаются в панели на сервере)
* Работа с правами юзера (в автозагрузке HKCU)
* При инсталле поднимается только один процесс
* Мобильные версии клиента (Android/iOS)
* Склейка с любым файлом (при запуске .exe .scr .com будет стартовать Ваш файл и дроппер)
* Возможность отключения UAC (win 7, 8, 8.1, 10)
* Языки клиента: Russian, English, Arabic, Chinese, Czech, Danish, Dutch, French, German, Hebrew, Italian, Japanese, Korean, Norwegian, Polish, Portuguese-Brazilian, Spanish, Swedish, Turkish
Функционал:
+Управление и просмотр экраном (отображает видео, настройки качества)
+Поддержка Aero в Windows 7 (высокое качество отображения)
+Поддержка мультимониторных систем (можно выбирать юзера за которым следить, если их несколько)
+Захват вэбкамеры (если их несколько можно выбрать) и захват звука (колонок\микрофона) (из минусов, нужны права админа)
+Командная строка (CMD консоль)
+Менеджер файлов (upload/download/delete/create)
+Запуск файлов
+Захват, передача буфера обмена
+Диспетчер задач с возможностью завершать службы, процессы и т.д.
+Редактор реестра
+Инвентаризация (сбор полных сведений о системе и железе)
+Запись экрана по расписанию
+Послать сообщение на рабочий стол
+Выключение, перезагрузка пк, спящий режим, отключение монитора.
+Блокировка клавиатуры, экрана
+Run-time результат (dropper): dyncheck Run-time результат (rdp module): dyncheck
+Чистота поддерживается постоянно.
+Чат

                                                 В комплект входит: Билд, Админка (Viewer), Мануал по использованию , Документация продукта.
                                                 Всё передается в настроенном и готовом для работы виде.

                                                 Вам не нужно самим ставить софт на сервер и т.д.
